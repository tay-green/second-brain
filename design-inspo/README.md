# Design Inspiration Repository

A curated collection of design inspiration organized by category with rich metadata for easy searching and reference.

## Structure

```
design-inspo/
├── ui-components/     - Buttons, forms, cards, navbars, modals, etc.
├── layouts/           - Grid systems, hero sections, page templates
├── color-palettes/    - Color schemes, gradients, themes
├── typography/        - Font pairings, text styles, headings
├── animations/        - Transitions, micro-interactions, motion design
├── patterns/          - Design patterns, best practices
└── inspiration/       - General ideas and concepts
```

## Metadata Format

Each inspiration item includes:
- **Title**: Descriptive name
- **Category**: Primary classification
- **Tags**: Searchable keywords
- **Source**: Attribution/URL
- **Date Saved**: Timestamp
- **Colors**: Color palette (if applicable)
- **Notes**: Personal thoughts and analysis

## Usage

Browse by category or search the `index.md` file for specific tags and keywords.
