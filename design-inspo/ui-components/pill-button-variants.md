# Pill-Shaped Button Component Variants

![Pill Button Variants](pill-button-variants.png)

## Overview
Three beautifully crafted pill-shaped button/label components showcasing different color treatments and subtle depth effects. Perfect example of modern, minimalist button design.

## Key Design Elements
- **Pill shape** with fully rounded corners (border-radius: 50% of height)
- **Three variants**:
  - Dark (black background, white text)
  - Light (white background, black text, subtle shadow)
  - Warm/Neutral (cream/beige background, brown text, gold border)
- **Consistent typography** - Clean sans-serif, medium weight
- **Subtle shadows** - Soft drop shadows for depth
- **Border treatment** - Thin outlined border on the warm variant

## Notable Features
- Consistent sizing across all variants
- Perfect visual balance between padding and text
- Subtle elevation through shadows
- Accessible contrast ratios
- Clean, modern aesthetic

## Design System Applications
- Primary/secondary/tertiary button styles
- Tag/label components
- Pill navigation
- Status badges
- Filter chips

- Design system button variants
- Tag/category labels
